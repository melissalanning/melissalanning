Finding National Airing Views

This example shows how to use the National Airing Views endpoint to find ratings of shows airing on NBC between November 5, 2018 and November 8, 2018.

Request

{

"group_by": [

"national_airdatetime",

"series_no"

],

"fields": [

"series_name",

"rating"

],

"filters": [

[

"eq",

"network_no",

5

],

[

"gte",

"day",

"2018-11-05"

],

[

"lt",

"day",

"2018-11-08"

]

]

}
