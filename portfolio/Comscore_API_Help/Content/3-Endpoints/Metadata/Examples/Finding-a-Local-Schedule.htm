Finding a Local Schedule

The following example shows how to use the Metadata endpoint (/tv/v4/metadata/local_schedule) to find local television schedules.

NOTE: At least one 'eq', 'in', or 'range' date filter is required to limit the number of returned schedule rows.

Request

{

"fields": [

"airdatetime",

"series_name",

"episode_title",

"network_name",

"station_no",

"station_name",

{

"alias": "call_sign",

"field": "station_call_sign"

},

{

"alias": "location",

"field": "station_location"

},

{

"alias": "tv_market_no",

"field": "station_primary_market_no"

},

{

"alias": "network_no",

"field": "station_affiliated_network_no"

}

],

"filters": [

[

"range",

"airdatetime",

"2018-01-01T00:00:00",

"2018-01-02T00:00:00"

],

[

"eq",

"station_no",

4201

]

]

}
