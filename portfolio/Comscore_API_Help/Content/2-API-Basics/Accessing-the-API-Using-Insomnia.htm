Accessing the API using Insomnia

Insomnia is a free and open source user-friendly desktop app that allows simple and efficient access to explore and evaluate the API prior to developing an automated solution.

Supported Systems:

• Linux
• MacOS
• Windows

To use the Insomnia application with the Comscore API, use the rapids_auth plugin we have developed that mimics the required authentication protocol.

Installing

  1. Download Insomnia: https://insomnia.rest/.
  2. Download and unzip the Comscore rapids_auth plugin.
  3. Run the Insomnia install wizard.
  4. Open Insomnia, click on Insomnia in the upper left corner, then Preferences.
  5. Click Plugins > Show Plugins Folder
  6. Copy the unzipped rapids_auth folder downloaded in step 2 into the plugins folder.
    NOTE: In some cases, when you un-zip a file in Windows an extra folder is created. The rapids_auth folder should have 6 files/folders in it.
  7. Close the window that opened for you to copy over the plugin.
  8. In the Insomnia client plugin manager, refresh the available plugins by clicking Reload Plugin List.
    The rapids_auth plugin should be displayed in the list.

Setting Up Your Environment to Connect to the Comscore Linear API

  1. In the Insomnia REST client, create an environment by clicking Insomnia > No Environment > Manage Environments in the upper-left corner.
  2. Click the plus (+) next to Sub Environments > Environment to add a new environment.
  3. Double-click New Environment at the top of the window to name your environment. Suggested Environment Name: Comscore Linear API
  4. Copy and paste the following code into the body, using your client_id and client_secret

    5. {

    "CLIENT_ID": "<your-client-id>",

    "CLIENT_SECRET": "<your-client-secret>"

    }

  5. Click Done to close the window.
  6. Select the environment you just created by clicking No Environment again and selecting the environment.
    You are now ready to use Insomnia to make calls to the Comscore Linear API.

    Note: Each time you make an API request, Insomnia will automatically add the appropriate headers to your request to authenticate.
    Example headers:
    Date: Wed, 16 Jan 2019 15:26:07 GMT
    Authorization: RTK <client-id>:<generated-signature>
